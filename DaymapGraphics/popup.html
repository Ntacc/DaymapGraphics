Hello!

Are you wondering why Daymap does not look any different? Open up the nav menu in Daymap and hover over the tools section. Then open up the item in the menu that drops down and enjoy!


Are you looking for a more in-depth guide? If you want that, here it is!

Opacity

This slider allows you to control the opacity of the foreground of Daymap. Note that this does not change the opacity of the text.

Blur

This changes how much the foreground "cards" blur the background when they are translucent or transparent. This is measured in pixels.

Auto attendance rainbow

One of the first features of Daymap Graphics was being able to click on the attendance indicator to change its colour. An easter egg was typing in "RAINBOW" as the colour would make it continuously change colour. Now, you can automate that process by checking this box.

Auto background

Check this box to make Daymap Graphics automatically set Daymap's background to be what you specify in the below text input.

Original layout

Revert the layout of Daymap pages to the layout from before the 30th of May 2022. (Personally, I recommend enabling this, as it fixes one of the main complaints with the new layout).

Dark mode

In the old layout of Daymap, there was no way to turn on dark mode. This allows turning on dark mode without needing the new layout. This also makes the text lighter, which can be useful if using a darker background.

Background CSS property

Out of all the features, this is the hardest to use. This should take the form of a CSS background property. You can use colours, linear gradients, and more. I personally use linear-gradient(to bottom right, #ABCDEF, turquoise, plum). However, you can also use images, including animated ones. The format for this should resemble url('https://i.ytimg.com/vi/dQw4w9WgXcQ/hqdefault.jpg').
