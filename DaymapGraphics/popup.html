Hello!

Are you wondering why Daymap does not look any different? Open up the nav menu in Daymap and hover over the tools section. Then open up the item in the menu that drops down and enjoy!


Are you looking for a more in-depth guide? If you want that, here it is!

Opacity

This slider allows you to control the opacity of the foreground of Daymap. Note that this does not change the opacity of the text.

Blur

This changes how much the foreground "cards" blur the background when they are translucent or transparent. This is measured in pixels.

Auto indicator rainbow

One of the first features of Daymap Graphics was being able to click on an indicator to change its colour. An easter egg was typing in "RAINBOW" as the colour would make it continuously change colour. Now, you can automate that process by checking this box.

Original layout

Revert the layout of Daymap pages to the layout from before the 30th of May 2022. (Personally, I recommend enabling this, as it fixes some of the main complaints with the new layout).

Dark mode

In the old layout of Daymap, there was no way to turn on dark mode. This allows turning on dark mode without needing the new layout. Turning on and off dark mode normally also is sometimes broken. This also makes the text lighter, which can be useful if using a darker background.

Background CSS property

Out of all the features, this is the hardest to use. This should take the form of a CSS background property. You can use colours, linear gradients, and more. I personally like linear-gradient(to bottom right, #ABCDEF, turquoise, plum). However, you can also use images, including animated ones. The format for this should resemble url('https://i.ytimg.com/vi/dQw4w9WgXcQ/hqdefault.jpg'). You may also wish to resize and reposition an image, so use the format url('https://www.nasa.gov/sites/default/files/thumbnails/image/main_image_star-forming_region_carina_nircam_final-5mb.jpg') 0 0 / cover.

Rainbow speed

Pretty self explanatory. If indicator rainbow is on, this just changes how fast the rainbow changes colour and moves around.

Profile picture URL

In feed view, there's a profile picture thingy that doesn't seem to get used. So if you want to change it, paste in a link to an image - usually I find the best way is by sending something on Discord then pressing "Copy Link" to get a link.
